Fruit-house-api说明指南

必备开发软件以及工具

1.IDEA (集成开发平台)
2.Navicat Premium (数据库可视化工具)
3.一台服务器或者前后端在一个局域网内开发 (后面有说明)
4.putty64/xshell
5.JDK
6.postman

综述

后端框架采用Spring Boot + Spring Mvc + Mybatis

源码

在工作区创建一个文件夹进入，鼠标右键，选择 Git Bash Here。

git clone https://github.com/Assisiwjj/Fruit-house-api.git

dao为Mybatis接口文件
domain为数据库实体类
service用于写具体的业务逻辑
web为controller层

第一行的jdbc:mysql://loaclhost:3306/将localhost改为数据库ip地址
第二行的root为数据库用户名
第三行为数据库密码
